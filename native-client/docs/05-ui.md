# Phase 5 — UI Widgets

## Overview

Port the 4 Mantine component panels + 1 modal to Material 3 Flutter widgets. The layout matches the web client: left sidebar (server stats, session list) and main content area (active session panel).

## 5.1 Theme & Scaffold

### Material 3 theme

```dart
// app.dart

  Widget build(BuildContext context) {
```

### Responsive layout

The web client uses a 4/8 grid (sidebar / main). Flutter equivalent using `LayoutBuilder`:

```dart
class HomePage extends ConsumerWidget {
  @override
  Widget build(BuildContext context, WidgetRef ref) {
    return Scaffold(
      appBar: AppBar(
        title: const Text('Sermon Translate'),
        actions: [
          IconButton(
            icon: const Icon(Icons.brightness_6),
            onPressed: () => /* toggle theme */,
          ),
        ],
      ),
      body: LayoutBuilder(
        builder: (context, constraints) {
          if (constraints.maxWidth >= 900) {
            return Row(
              crossAxisAlignment: CrossAxisAlignment.start,
              children: [
                SizedBox(width: 360, child: _sidebar(ref)),
                Expanded(child: _main(ref)),
              ],
            );
          }
          return SingleChildScrollView(
            child: Column(
              children: [_sidebar(ref), _main(ref)],
            ),
          );
        },
      ),
    );
  }
}
```

Breakpoint at 900px matches the web client's `md` (Mantine's medium breakpoint is 992px; 900 is close enough for native where window sizes differ).

## 5.2 Server Stats Panel

**Replaces:** `ServerStatsPanel.tsx`

```
┌─────────────────────────────┐
│ Server                      │
│ ⏱ Uptime: 2h 15m 30s       │
│ ⚡ Sessions: 1 active / 5   │
│ 💾 Processed: 12.3 MB       │
│ 🔧 Pipelines: 3             │
└─────────────────────────────┘
```

Implementation: `Card` containing a `Column` of `ListTile` widgets with `leading` icons.

```dart
class ServerStatsPanel extends ConsumerWidget {
  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final statsAsync = ref.watch(serverStatsProvider);

    return Card(
      child: Padding(
        padding: const EdgeInsets.all(16),
        child: statsAsync.when(
          data: (stats) => stats == null
              ? const Text('Connecting to server...')
              : Column(
                  crossAxisAlignment: CrossAxisAlignment.start,
                  children: [
                    Text('Server', style: Theme.of(context).textTheme.titleMedium),
                    const SizedBox(height: 8),
                    _row(Icons.schedule, 'Uptime: ${_fmt(stats.uptimeSeconds)}'),
                    _row(Icons.bolt, 'Sessions: ${stats.activeSessions} active / ${stats.totalSessions}'),
                    _row(Icons.storage, 'Processed: ${_bytes(stats.totalBytesProcessed)}'),
                    _row(Icons.settings, 'Pipelines: ${stats.availablePipelines}'),
                  ],
                ),
          loading: () => const Center(child: CircularProgressIndicator()),
          error: (_, __) => const Text('Server offline'),
        ),
      ),
    );
  }
}
```

Implementation: `Card` with header `Row` (title + `IconButton`), then `ListView.builder` of tappable session `Card`s with status `Chip`.

Status color mapping (same as web):

```dart
Color statusColor(SessionStatus status) => switch (status) {
  SessionStatus.created => Colors.blue,
  SessionStatus.active  => Colors.green,
  SessionStatus.paused  => Colors.amber,
```

Tapping a session card sets `selectedId` (local state). The `[+]` button opens the New Session dialog.

## 5.4 New Session Dialog

**Replaces:** `NewSessionModal.tsx`

```
┌───────────────────────────────┐
│ New Session                   │
│                               │
│ Pipeline:  [Whisper TTS    ▼] │
│ Transcribes speech to text.   │
│                               │
│ Label:     [________________] │
│                               │
│ ┌──────────┐┌──────────────┐  │
│ │ 🎤 Mic   ││ 📁 MP3 File  │  │
│ └──────────┘└──────────────┘  │
│                               │
│ Input:  [Built-in Mic    ▼]   │
│   — or —                      │
│ [Choose MP3 file]             │
│ test-speech.mp3               │
│                               │
│ Output: [Speakers        ▼]   │
│                               │
│         [Cancel] [Start]      │
└───────────────────────────────┘
```

Implementation: `showDialog` returning `AlertDialog` with `SingleChildScrollView` body.

Key widgets:
- **Pipeline**: `DropdownButtonFormField<String>` populated from `fetchPipelines()`
- **Source toggle**: `SegmentedButton<AudioSourceType>` with `mic` and `file` segments
- **File picker**: `ElevatedButton` using `FilePicker.platform.pickFiles(type: FileType.custom, allowedExtensions: ['mp3'])`
- **Input device**: `DropdownButtonFormField` from `listInputDevices()` (hidden when source is `file`)
- **Start button**: disabled when source is `file` and no file selected

On submit:
1. Call `apiClient.createSession(SessionCreate(...))`
2. Create `MicSource(deviceId)` or `FileSource(filePath)`
3. Call `ref.read(activeSessionProvider.notifier).start(session, source)`
4. Close dialog

## 5.5 Active Session Panel

**Replaces:** `ActiveSessionPanel.tsx`

```
┌─────────────────────────────────────────┐
│ Active Session                 🟢 Live  │
│                                         │
│ Session:  abc123def456                  │
│ Pipeline: whisper-tts                   │
│ Duration: 45s                           │
│ Audio In:  384.0 KB (47 chunks)         │
│ Audio Out: 0 B (0 chunks)              │
│ Latency:   5000ms                       │
│                                         │
│ Transcript                              │
│ ┌─────────────────────────────────────┐ │
│ │ The grace of God is sufficient for  │ │
│ │ all.                                │ │
│ │                                     │ │
│ └─────────────────────────────────────┘ │
│                                         │
│                            [🔴 Stop]    │
└─────────────────────────────────────────┘
```

Implementation: `Card` with:
- Header `Row`: title + `Chip(label: connected ? 'Streaming' : 'Disconnected')`
- Stats: `Column` of `Text` widgets (only shown when `liveStats != null`)
- Transcript: `ListView.builder` inside a `SizedBox(height: 200)` with `ScrollController` that auto-scrolls to bottom
- Stop button: `OutlinedButton` calling `ref.read(activeSessionProvider.notifier).stop()`

```dart
class ActiveSessionPanel extends ConsumerWidget {
  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final stream = ref.watch(audioStreamProvider);
    final active = ref.watch(activeSessionProvider);
    if (active == null) return const SizedBox.shrink();

    return Card(
      child: Padding(
        padding: const EdgeInsets.all(16),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            // Header row with title and connection badge
            Row(
              mainAxisAlignment: MainAxisAlignment.spaceBetween,
              children: [
                Text('Active Session',
                    style: Theme.of(context).textTheme.titleMedium),
                Chip(
                  label: Text(stream.connected ? 'Streaming' : 'Disconnected'),
                  backgroundColor: stream.connected ? Colors.green : Colors.grey,
                ),
              ],
            ),
            const SizedBox(height: 8),

            // Session info
            Text('Session: ${active.session.id}'),
            Text('Pipeline: ${active.session.pipelineId}'),

            // Live stats (when available)
            if (stream.liveStats != null) ...[
              const SizedBox(height: 8),
              Text('Duration: ${stream.liveStats!.durationSeconds.toStringAsFixed(0)}s'),
              Text('Audio In: ${_bytes(stream.liveStats!.bytesReceived)}'),
            ],

            // Transcript
            if (stream.transcript.isNotEmpty) ...[
              const SizedBox(height: 12),
              Text('Transcript',
                  style: Theme.of(context).textTheme.titleSmall),
              const SizedBox(height: 4),
              SizedBox(
                height: 200,
                child: ListView.builder(
                  controller: _scrollController,
                  itemCount: stream.transcript.length,
                  itemBuilder: (_, i) => Text(
                    stream.transcript[i],
                    style: Theme.of(context).textTheme.bodySmall,
                  ),
                ),
              ),
            ],

            // Stop button
            const SizedBox(height: 12),
            Align(
              alignment: Alignment.centerRight,
              child: OutlinedButton.icon(
                onPressed: () =>
                    ref.read(activeSessionProvider.notifier).stop(),
                icon: const Icon(Icons.stop),
                label: const Text('Stop'),
                style: OutlinedButton.styleFrom(
                    foregroundColor: Colors.red),
              ),
            ),
          ],
        ),
      ),
    );
  }
}
```

## 5.6 Widget ↔ Provider Wiring Summary

| Widget | Watches | Actions |
|---|---|---|
| `ServerStatsPanel` | `serverStatsProvider` | — |
| `SessionListPanel` | `sessionsProvider` | `onNew` → open dialog |
| `NewSessionDialog` | `fetchPipelines()` (one-shot) | `activeSessionProvider.start()` |
| `ActiveSessionPanel` | `activeSessionProvider`, `audioStreamProvider` | `activeSessionProvider.stop()` |
| `HomePage` | `activeSessionProvider` (to show/hide panel) | — |

- Starting a mic session shows active panel with live stats
